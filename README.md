# OkShell

![logo](docs/assets/logo.svg)

---

OkShell is a customizable desktop shell for Hyprland.  It is built in Rust using [Relm4](https://relm4.org/) and [Wayle-services](https://github.com/wayle-rs/wayle-services).

### Features

- Theming system backed by Matugen with built-in dynamic and static themes
- Wallpaper management
- Customizable bars and menus
- Screenshot / screen recording tool
- Lock screen
- Polkit agent
- Clipboard manager
- Notifications
- Built-in settings UI

And many more!
