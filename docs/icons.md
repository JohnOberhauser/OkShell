# Icons

There are two types of icon themes for OkShell.
`Shell` icons and `App` icons.

Go to `Settings` -> `Theme` -> `Icon Theme` to change your themes.

`App` icon themes come from your normal installed icon themes.
You can see these used in the dock and app launcher.

`Shell` icon themes are all the other icons you see.
There are a couple of built-in `Shell` icon themes you can choose from.
You can also make your own `Shell` icon theme.

## Making your own Shell icon theme or changing an icon

To create your own theme, start by copying one of the [existing built-in themes](https://github.com/JohnOberhauser/OkShell/tree/main/icons).
Rename the directory and change the theme name in the `index.theme` file.
Then change any icons you want.
After that, move your new icon theme to `~/.config/okshell/icons/`.

Then you can reopen settings, and you will see your new icon theme in the `Shell` icon theme dropdown.
